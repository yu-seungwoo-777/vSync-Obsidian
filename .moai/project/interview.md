# Project Interview

## Round 1: Vision
Question: Vector의 핵심 비전은 무엇입니까?
Answer: 동기화 서버 + 플러그인 (PostgreSQL 기반 셀프호스팅 동기화 서버 + Obsidian 플러그인 클라이언트. 다중 디바이스 실시간 동기화, LLM Tool Use로 Vault 접근 지원)

## Round 2: Technology
Question: 기술 스택을 확인합니다. PRD에 정의된 스택을 그대로 사용할까요?
Answer: PRD 스택 그대로 (Node.js 22 LTS + Fastify 5 + Drizzle ORM + PostgreSQL 16 + MinIO)
